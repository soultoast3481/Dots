# Dots
**A SIMPLE HTML/PYTHON app that runs on your WIFI**

Dots is a real-time, multiplayer game using Flask and SocketIO. Players move around the canvas and leave fading trails as they go!

## Features
- Real-time player movement and trails
- Unique colors for each player
- Easy setup and deployment

## Installation (Windows)
1. **Clone the repository**:
    ```bash
    git clone https://github.com/soultoast3481/Dots.git
    cd dots
    ```

2. **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3. **Run the server**:
    ```bash
    python app.py
    ```

4. **Open the game**:
    - Go to `http://localhost:5000` in your browser.

## How to Play
- Move your mouse to control your dot.
- See your trail and other players in real-time!
